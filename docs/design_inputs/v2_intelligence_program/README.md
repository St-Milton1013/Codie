## Program classification

The inputs belong to three related but distinct tracks:

1. Relationship Intelligence is Class 2 measured evidence. It remains a
   separate core V2 program and may later provide evidence references to Jin.
2. Rules, model profiles, the Correction Ledger, the Theory Corpus, the fixed
   Jin regression corpus, and Jin orchestration form a dependency-ordered
   intelligence program.
3. The Decision Evidence Panel, Jin chat experience, and Knowledge Vault are
   downstream presentation and export programs. They must consume stable
   packets and may not reproduce evidence, recommendation, or correction logic.

The consolidation and sequencing rules are recorded in:

`docs/ROADMAP_PATCH_V2_INTELLIGENCE_PROGRAM_INTAKE.md`
